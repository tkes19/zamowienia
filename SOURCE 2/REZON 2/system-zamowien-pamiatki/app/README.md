# System Zarządzania Zamówieniami Pamiątek

> Szczegółowa dokumentacja znajduje się w katalogu `docs/`.

## Skrót projektu

- Aplikacja Next.js 14 + TypeScript dla zarządzania zamówieniami personalizowanych produktów.
- Role: `ADMIN`, `SALES_REP`, `WAREHOUSE_STAFF`, `CUSTOMER` (w planie).
- Backend: API Routes + Prisma + Supabase.

## Start lokalny

```bash
yarn install
yarn dev
```
